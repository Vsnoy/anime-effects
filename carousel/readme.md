# 轮播动效

## 场景

- 搜索热词轮播动效
- 气泡提醒轮播动效
- 热门书籍轮播动效
- 卡片翻转轮播动效

## 要点

- 自动轮播
- 手动轮播
- 悬停暂停轮播
- 头部多插一张尾部图片，尾部多插一张头部图片，为了首尾丝滑轮播切换
- 节流时间要大于等于过渡时间，过渡动画未完成，限制手动切换轮播图，且给首尾边缘处理留足时间
